自监督学习

多模态常用的自监督学习方法：

  1. 对比学习（Contrastive Learning）

对比学习是一种常用的自监督学习框架，通过最小化来自不同模态的匹配数据的距离，同时最大化不匹配数据的距离来学习通用的特征表示。

CLIP (Contrastive Language-Image Pretraining)：

CLIP 是 OpenAI 提出的基于对比学习的多模态模型。它通过同时对图像和文本进行编码，最大化图像与描述它的文本的相似性，同时最小化其他图像和文本的相似性。 CLIP 利用大量的图像-文本对进行训练，使得模型能够在图像和文本之间建立语义联系。它将图像和文本映射到共享的特征空间中，学会匹配对应的模态表示。 VSE++ (Visual-Semantic Embedding)：

视觉-语义嵌入是一种通过对比学习的方式，将图像和文本的特征表示映射到同一个共享空间中。模型训练的目标是最大化图像-文本对之间的相似度，同时最小化无关的图像-文本对的相似度。

  1. 互信息最大化（Maximizing Mutual Information）

互信息最大化通过最大化来自不同模态的表示之间的互信息，以促进跨模态的特征学习。

DCCA (Deep Canonical Correlation Analysis)：

DCCA 是一种基于深度学习的对齐方法，用于最大化两种模态之间的相关性。它通过学习将不同模态的数据映射到一个共享的潜在空间，使得映射后的表示具有较高的互信息和相关性。 InfoNCE Loss：

InfoNCE 是一种常用于自监督学习的损失函数，它通过最小化不同模态表示之间的负对数相似性，最大化匹配样本之间的互信息。

  1. 伪标签生成（Pseudo-labeling）

在多模态数据中，可以通过一种模态的数据生成伪标签，来监督另一种模态的学习。

DeepCluster：

DeepCluster 使用聚类方法将无标签的数据划分为不同的类，然后利用这些伪标签进行监督学习。在多模态任务中，可以对一种模态的表示进行聚类，然后将聚类结果作为另一种模态的监督信号。 Cycle-Consistency：

循环一致性是一种生成伪标签的方式，例如在视觉-语言任务中，通过一个编码器将文本转化为图像，然后再从图像重新生成文本。这个循环过程的目标是让重新生成的文本与原文本一致，以此作为监督信号。 4. 多模态对齐（Multimodal Alignment） 通过对齐不同模态之间的信息，模型可以学到不同模态之间的关联关系。

BERT-Style Masking（BERT 类遮掩学习）：

多模态 BERT 使用遮掩的方法来学习文本和其他模态（如图像、音频）之间的关联。例如，VideoBERT 和 VisualBERT 等模型通过遮掩输入中的某些模态（例如，遮掩文本中的词或视频帧）并让模型进行预测，从而学习跨模态表示。 Cross-Modal Matching：

在跨模态匹配任务中，模型尝试预测不同模态之间是否匹配。例如，在视觉-语言任务中，模型给定一张图片和一个句子，通过判断它们是否匹配，来对图像和文本之间的关系进行建模。

  1. 自回归建模（Autoregressive Modeling）

自回归建模是一种通过序列生成来学习多模态关系的方法，通常用于多模态生成任务。

GPT-3 风格的多模态模型（如 Flamingo）：

这类模型结合了文本和视觉信息，通过自回归生成文本的方式来学习图像和文本之间的关联。模型会使用来自视觉模态的特征作为输入条件，生成描述性的文本或者回答与图像内容相关的问题。

VQ-VAE（Vector Quantized Variational Autoencoder）：

VQ-VAE 可以用于多模态的生成任务，通过将不同模态的输入进行量化，然后让模型在量化后的离散空间中学习。这种方法也可以用来对不同模态进行联合建模。

  1. 生成对抗网络（Generative Adversarial Networks, GANs）

GANs 在多模态自监督学习中也有广泛应用，尤其是在生成任务中，如图像生成和文本生成。

CycleGAN for Cross-Modal Generation：

CycleGAN 可以应用于不同模态之间的生成转换，例如将图像转换为文本描述，或者从文本描述生成相应的图像。它通过循环一致性损失，确保从一种模态生成的另一种模态能再次还原到原始模态。

CLIP-Guided GANs：

结合 CLIP 的对比学习表示和 GAN 的生成能力，可以实现跨模态的生成和控制。CLIP 的视觉和语言特征可以用于指导 GAN 的生成，以确保生成内容与给定的文本描述匹配。

  1. 视觉-语言预训练模型（Vision-Language Pre-training Models）

近年来，基于 Transformer 的视觉-语言预训练模型在多模态自监督学习中取得了显著的进展。

VL-BERT, VisualBERT, ViLBERT：

这些模型通过在大规模视觉-语言数据上预训练，学习不同模态之间的交互特征。训练任务通常包括图像-文本匹配、掩蔽语言建模等，帮助模型学习视觉和语言模态的融合表示。 ALIGN：

ALIGN 是一种端到端的多模态对齐模型，通过对图像和文本对进行对比学习，学习它们在共享表示空间中的嵌入。ALIGN 使用了大量的未标注图像-文本对进行训练，能够有效地捕捉两种模态之间的语义联系。

CLIP

https://zhuanlan.zhihu.com/p/646790176 1. 用图像编码器把图像编码成向量 a； 2. 用文本编码器把文本编码成向量 b； 3. 计算 a·b， 4. 如果 a 和 b 来自一对儿配对的图和文字，则让 a·b 向 1 靠近； 5. 如果 a 和 b 来自不配对儿的图和文字，则让 a·b 向 0 靠近；

选择了对比学习的方法，也就是本篇CLIP文章所讲的方法 - 预训练：预训练方法也不难理解，模型由两个编码器组成Image-encoder (ViT / ResNet) & text-encoder (transformer)，如上图左半部分，左上紫色的梯形是文本编码器，左下绿色的梯形是图像编码器。作者收集了大量匹配的图像和文本对，然后分别走各自的编码器得到各自的特征，再计算特征两两之间的cos相似度，让配对的特征相似度越近越好，不配对的相似度越远越好。这样就可以完成了CLIP的与训练。

  • 图像分类任务：如上图右半部人，给一堆类别名，把类别名填到“A photo of a {object}.”里面然后编码。然后图像也做编码，编码完成后拿图像特征跟文本特征比cos相似度，跟哪个特征距离最近，我们就认为模型把这张图分到了哪个类别里，so easy~~~

  • 再说一下数据：收集了400 million 的数据文本对，跟webText差不多，称为WIT，即WebImageText。

  • 然后是两个编码器模型：作者尝试了ResNet、EfficientNet、Transformer、最大用了ViT。最小与最大模型参数量为1:100。

BLIP

https://zhuanlan.zhihu.com/p/663378531

任务

  1. 图文对比任务ITC，图像encoder分支采用ViT结构，文本encoder部分采用BERT结构，[CLS] token用来代表整个句子的特征。图像特征和文本特征进行对比损失，从而进行对齐操作。
  2. 图文匹配任务ITM，采用image-grounded text encoder，具体结构也是BERT，采用cross-attention来融合图像特征和文本特征，相当于一个多模态的encoder，[Encode] token用来代表多模态融合后的特征，最后在该特征后面增加一个mlp分类head，区分图文是否匹配。
  3. 语言模型任务LM，采用image-grounded text decoder，采用causal self-attention来编码图像特征和文本特征。与BERT的mask机制不同，这里causal self-attention是一种序列注意力机制，相当于预测接下来的语句。[Decode] token被用来表示句子的开头。因此该语言模型任务能够为图像生成一段话caption，为后续的captioner、filter机制奠定基础。

模型

采用 对比学习 的方式，基本属于自监督了

  • Image Encoder (ViT) ：首先进行图像特征的提取，这个前向过程相对较重；
  • Text Encoder (BERT) ：这是一个标准的 BERT，这里用 ITC (Image-Text Contrastive Loss) 目标函数激活 Text Encoder 模块，目标是对齐 Image Encoder Transformer 和 Text Encoder Transformer 的特征空间；
  • Image-grounded Text Encoder (变种 BERT)：在标准 BERT 的结构里，于 Bi Self-Att 和 Feed Forward 之间插入 Cross Attention (CA) 模块，以引入视觉特征。这里用 ITM (Image-Text Matching Loss) 目标函数激活，ITM 是一个二分类任务，用来预测 image-text pair 的 正匹配 还是 负匹配，目的是学习 image-text 的多模态表示，调整视觉和语言之间的细粒度对齐；
  • Image-grounded Text Decoder (变种 BERT)： 将 Image-grounded Text Encoder 结构中的 Bi Self-Att 替换为 Causal Self-Att。这里用 Language Modeling Loss (LM) 目标函数激活，目标是生成给定图像的文本描述。 ###

BLIP2

特点

  • 采用已经训练好的单模态模型，从而避免train-from-scratch，减少训练成本；
  • 将单模态模型的参数进行冻结，从而避免灾难性遗忘问题，充分利用已经训练好的单模态模型；
  • 利用Q-Former来对图文进行对齐，从而让图像和文本产生交互 https://zhuanlan.zhihu.com/p/664601983 ### 模型
  1. Image encoder
  2. Q-Former
  3. LLM 在冻结的视觉模型和大语言模型间进行视觉-语言对齐。 Q-Former主要参考了Bert的enconder，总共12层，偶数层增加了一个ca层，奇数层不增加。Q-Former通过bert-base预训练模型初始化，SA是共享权重的，CA随机初始化，CA层只对图像起作用。图中颜色相同代表权重共享。

图像特征提取用vit，使用了ViT-L/14 from CLIP、ViT-G/14 from EVA-CLIP预训练模型，移除ViT最后一层，使用倒数第二层特征。这里vit图像输入的分辨率是224，可以考虑换成sam的vit，输入分辨率是1024，可以提取更丰富的图像特征。

设计了一个可学习的Query embeddings作为输入，大小为【b,32,768】，query通过SA(self attention)进行自我交互（让每个token学习自己应该关注什么），学习从图片里边提取哪些token；通过CA(cross attention)和图像特征进行交互，进行decoder，同时和文本分支的sa共享权重，实现了和文本进行交互。

利用Q-Former来对图文进行对齐，从而让图像和文本产生交互；用Q-Former学习的文本和图像向量，加上一个全连接层（一个Linear，从768维到2560维），然后输入到大语言模型，预测文本输出。 ## 多模态

ALBEF

三个任务

https://zhuanlan.zhihu.com/p/652928157

3个任务：image-text contrastive learning(ITC)、masked language modeling(MLM)和image-text matching(ITM)。3个loss：TC图文损失利用的标准的InfoNCE损失；ITM图文匹配损失的正样本对是原始输入图文对，而负样本对是挖掘出来的不匹配的图文对；MLM文本掩码损失输入的文本是重新进行mask操作的，与ITC中的完整文本是不相同的。

  • ITC：ITC图文对比损失，采用的是标准的InfoNCE loss，只是在构建真值时，采用了蒸馏部分的特征和队列中的特征的相似度作为target。相当于有两个image encoder和两个text encoder，其中一个采用动量更新参数，并构建图像特征和文本特征，并存入队列中，从而增大对比损失时的负样本数量。
  • ITM图文匹配损失，包含了图文对匹配部分和图文对不匹配部分。匹配的图文对是直接采用的原始图文对。而不匹配的图文对是利用图文相似度来进行挖掘的。
  • MLM文本掩码损失：输入input_ids和image_embeds，随机mask文本，利用cross-attention来融合mask后的文本和图像特征，输出被mask的token的概率logits。（单个token的logits是一个vocab_size大小的概率向量）
动量蒸馏

动量蒸馏法：

动量模型 = EMA(基础模型)  # 类似于EMA，权重移动平均；

伪标签 = 动量模型.forward(input)

动量loss = get_loss(模型预测结果，伪标签)；

loss = 原本的loss + 动量loss；
